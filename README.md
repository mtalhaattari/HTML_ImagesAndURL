# NAVTTC Advanced Web Development - Assignment 💻

## Project Overview 🚀
This repository contains a foundational web development assignment focusing on structured HTML, image implementation, and hyperlink navigation. The goal of this project was to demonstrate proficiency in linking internal and external resources while maintaining a clean directory structure.

## Core Features ✨
* **Structured HTML5:** Semantic usage of tags to ensure document clarity.
* **Image Integration:** Proper implementation of `<img>` tags with optimized `alt` attributes for accessibility.
* **Hyperlinks:** Navigation using both absolute and relative URLs to connect different pages and external resources.
